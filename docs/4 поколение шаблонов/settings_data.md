# settings_data.json

Дефолтные настройки виджета

## Текстовые поля

Мультиязычность реализуется через объект

```json
"title": {
  "ru":"Преимущества",
  "en":"Benefits",
  "ua":"Переваги",
  "es":"Beneficios"
}
```

## Поле файл

Указывается id файла из http://file-store.myinsales.ru/

```
"title": {
  "ru":"Преимущества",
  "en":"Benefits",
  "ua":"Переваги",
  "es":"Beneficios"
}
```

## Пример:
```
{
  "wide_background": false,
  "wide_content": false,
  "padding_top": 2,
  "padding_bottom": 2,
  "margin_top": 1,
  "margin_bottom": 1,
  "content_max_width": 1120,
  "benefit_row_gap": 2,
  "benefit_column_gap": 2,
  "benefit_img_border_radius": 5,
  "benefit_img_ratio": 1,
  "title": {
    "ru":"Преимущества",
    "en":"Benefits",
    "ua":"Переваги",
    "es":"Beneficios"
  }
}

```
